🎮 Tic Tac Toe Game (Tkinter)

A simple Tic Tac Toe game built using Python and Tkinter.

This project demonstrates how to create a graphical user interface (GUI) using Tkinter and implement basic game logic such as turn switching, win detection, and draw detection.

📌 Features

3x3 game board

Two-player mode (Player X vs Player O)

Automatic turn switching

Win detection (rows, columns, diagonals)

Draw detection

Buttons disabled after being clicked

Simple and clean GUI

🖥️ Requirements

Python 3.x

Tkinter (comes pre-installed with Python)
